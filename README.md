# Minecraft Lifesteal Plugin 1.21.10 [UPDATED]
This is a Lifesteal Plugin for Minecraft Java 1.21.10. It's nice and simple, to take a heart from someone you may hit them for a CHANCE to get a heart fragment. You need 8 heart fragments and a Totem of Undying to craft a new heart. If you kill someone they will respawn with 1 less heart and so on.

# NOTE
This is re-used code from a plugin I made back for Minecraft 1.18.2 so something may not work. If that's the case please contact me here.

# Useful Commands
/lifehelp | Gives detailed help and insight to the plugin.
/heart <giveheart>|<givefrags> | Gives you (only) a Fresh Heart or a Heart Fragment
